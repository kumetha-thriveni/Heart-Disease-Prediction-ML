Heart Disease Prediction using Machine Learning

>Project Overview

This project predicts whether a patient has heart disease based on clinical features using Machine Learning models.

>Dataset

Heart Disease Dataset from Kaggle

>Features Used

* Age
* Sex
* Chest Pain Type
* Resting Blood Pressure
* Cholesterol
* Max Heart Rate
* Oldpeak
* Thal

>Machine Learning Models

* Logistic Regression
* Random Forest
* XGBoost

>Evaluation Metrics

* ROC-AUC
* Classification Report

>Visualization

* Correlation Heatmap
* Feature Importance
* Interactive Dashboard

>Tools Used

* Python
* Google Colab
* Scikit-Learn
* XGBoost
* Plotly
* SHAP
